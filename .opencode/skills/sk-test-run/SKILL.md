테스트 프레임워크를 실행하고 결과를 수집하라.

## 실행 절차

1. 프로젝트의 빌드 시스템을 확인하여 테스트 실행 명령을 결정하라
2. 테스트를 실행하라
3. 커버리지를 측정하라
4. 실패한 테스트가 있으면 상세 정보를 수집하라

## 결과 수집 항목

- 총 테스트 수, 통과/실패/스킵 수
- 실패 테스트 목록 (테스트명, 실패 원인, 스택트레이스)
- 커버리지 비율 (라인/브랜치)
- 실행 시간

## 판정 기준

- 모든 테스트 통과 + 목표 커버리지 달성 → 성공
- 테스트 실패 있음 → 실패 (상세 원인 보고)
- 커버리지 미달 → 경고 (추가 테스트 필요)
